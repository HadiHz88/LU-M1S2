# IN411- Introduction to Big Data

## Description

This course aims to equip students with a solid foundation in Big Data concepts, models, tools,and platforms essential for handling large-scale datasets and analytical tasks. Students will gain anunderstanding of the Big Data ecosystem, including key technologies like Hadoop and ApacheSpark. The curriculum covers distributed file systems, MapReduce programming, an introductionto NoSQL databases, and modern data storage concepts like data lakes. Practical labs will providehands-on experience with setting up environments, processing data, and performing analyticsusing real-world tools.
