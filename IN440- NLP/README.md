# IN440- NLP (Natural Language Processing)

## Packages

### NLTK ( Natural Language Toolkit)

To download NLTK library, run the following command in your terminal:

```pip install nltk```

To download the needed NLTK data files in this course, you can use the following command in your Python code:

```python
import nltk
nltk.download('punkt')
nltk.download('punkt_tab')
nltk.download('wordnet')
```

Use this [link](https://www.nltk.org/index.html) for more information about NLTK.
