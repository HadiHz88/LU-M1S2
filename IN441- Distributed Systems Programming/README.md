# IN441- Distributed Systems Programming

## Description

This course provides a comprehensive understanding of the fundamental concepts and inherent challenges of distributed systems. Students will gain practical skills in implementing low-level networking using TCP and UDP, and design and build distributed applications leveraging various communication paradigms such as RMI, RPC, and gRPC. The curriculum also covers developing and consuming RESTful APIs for web-based services. Emphasis is placed on applying best practices for scalability, fault tolerance, and performance optimization, ensuring students can design, develop, and deploy robust and efficient distributed applications.
