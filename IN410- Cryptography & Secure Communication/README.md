# IN410- Cryptography & Secure Communication

## Description

This course provides a comprehensive understanding of applied cryptography and its role in securing modern communications. Students will delve into the core principles and practical applications of symmetric and asymmetric-key cryptography, cryptographic hash functions, and digital signatures. The curriculum also covers essential security protocols like TLS/SSL and IPsec, explores Public Key Infrastructure (PKI), and addresses wireless security standards. Emphasis is placed on understanding the mechanisms behind these technologies and their practical implementation to protect against common cryptographic attacks.
